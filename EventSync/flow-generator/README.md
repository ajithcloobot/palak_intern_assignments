# 🎬 Flow Generator

**AI-powered flow engine with automatic screen generation and professional video creation for any product workflow**

## ✨ Features

### 🎬 **Dynamic Video Generator**
- 🤖 **Auto-generate screens** for any product (Banking, E-commerce, Restaurant, etc.)
- 📱 **Realistic interfaces** (forms, dashboards, payment terminals)
- 🎥 **Professional video player** with play/pause/progress controls
- ⚡ **3-second transitions** between screens

### 🧠 **Intelligent Flow Engine**
- 🎯 **Behavior-based flow detection** with confidence scoring
- 🔍 **Pattern analysis** for user actions
- 📊 **Decision reasoning** display
- 🔄 **Adaptive flow visualization**

### 🌍 **Universal Flow Engine**
- 🏢 **Any product support** - Banking, E-commerce, Restaurant, SaaS, Gaming
- 📋 **Custom screen registry** with triggers and icons
- 🎯 **Journey step definition**
- 🤖 **Auto-detection** of common patterns

### 🎥 **Professional Video Demo**
- 📹 **High-quality animated walkthrough**
- 🏢 **Realistic mockups** (Google Forms, Excel, Gmail)
- ⌨️ **Typing animations** and data transfer effects
- 🎮 **Interactive video controls**

## 🚀 Quick Start

### **Method 1: Try the Dynamic Video Generator**
1. Open [`17-dynamic-video-generator.html`](17-dynamic-video-generator.html)
2. Enter product name (e.g., "Banking App")
3. Click "🤖 Auto-Generate Screens"
4. Add journey steps like "User logs into bank account"
5. Click "🎬 Generate Dynamic Video"
6. Press "▶️ Play" to watch your custom video!

### **Method 2: Use the Main Hub**
1. Open [`index.html`](index.html)
2. Choose any component from the dashboard
3. Follow the on-screen instructions
