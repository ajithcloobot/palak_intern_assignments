### Getting Started

#### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

#### Installation

1. Navigate to the project directory:
   ```bash
   cd "april task cloobot"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000`

#### Usage
1. Click the **"Start User Journey Flow"** button to begin the demonstration
2. Watch the automated progression through all steps
3. Use Pause/Resume controls during the flow
4. View the generated ticket at the end

## 🎫 User Journey Flow

1. **Registration Form** - User fills out registration details
2. **Google Form Submission** - Data sent to Google Forms (real/simulated)
3. **Google Sheets Update** - Admin sees registration in dashboard
4. **Admin Review** - Admin approves or rejects registration
5. **Email Notification** - User receives approval with ticket details
6. **Ticket Generated** - Final digital ticket created
